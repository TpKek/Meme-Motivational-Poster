# Meme Motivational Poster

A fun, minimalist motivational poster web page built with HTML and CSS. This project demonstrates centering techniques, typography, and creative use of Google Fonts.

## About

A simple yet effective motivational poster featuring a meme-style image with bold typography. The dark theme and centered layout create a striking visual impact.

## What I Learned

This project reinforced core CSS skills:

- **Centering Techniques**: Using `margin-left: 25%` and `width: 50%` for horizontal centering
- **Google Fonts Integration**: Importing and applying custom web fonts
- **Text Transform**: Using `text-transform: uppercase` for stylistic effect
- **Image Styling**: Adding borders with `border: 5px solid white`
- **Dark Theme Design**: Creating contrast with black backgrounds and white text
- **Responsive Images**: Using `width: 100%` for flexible image sizing

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/TpKek/Meme-Motivational-Poster.git
   cd Meme-Motivational-Poster
   ```

2. Open in browser:
   - Simply open `index.html` in any modern web browser

## Key CSS Techniques

```css
/* Horizontal centering */
div {
  width: 50%;
  margin-left: 25%;
}

/* Uppercase transformation */
h1 {
  text-transform: uppercase;
}

/* Image styling */
img {
  border: 5px solid white;
  width: 100%;
}
```

## Customization

To create your own poster:

1. Replace the image in `assets/images/`
2. Update the heading and paragraph text in `index.html`
3. Adjust colors in `style.css` if desired
